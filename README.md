# Cytoid_AI_charting

Game: https://cytoid.wiki/zh/
For educational purposes only — this is an undergraduate STATS 402 final project at DKU.
Data sources are documented in the code.

Many thanks to all the Cytoid charters for their hard work!
This is just a small experiment in AI-based chart generation.

I tried LSTMs, Transformers, and CNNs, and in the end CNNs worked best — probably just a hyperparameter-tuning issue. I’m not very familiar with these models yet, so I’m still unclear about many of the practical details.

My only real “contribution” is probably a preprocessing pipeline for the Cytoid input format (lol).

For more information about the game itself, see the official Cytoid wiki: https://cytoid.wiki/zh/
